## 🚀 Copilot-Driven Automated SDLC

This project demonstrates how GitHub Copilot can be used as an AI agent to automate the Software Development Life Cycle (SDLC).

### 🔄 Workflow
1. Requirements Generation (Business Analyst)
2. Architecture Design (Software Architect)
3. Design Review (Reviewer)
4. Implementation Planning (Tech Lead)
5. Code Generation (Engineer)
6. Code Review
7. Test Plan Generation

### 🧠 Key Concept
This project simulates MCP (Model Context Protocol) by chaining outputs between stages.

### 🛠️ Tech Stack
- Python
- GitHub API
- Markdown
- Pytest

### 📁 Structure
- `/src` → Implementation modules
- `/tests` → Test cases
- `/docs` → Generated documentation

### ✅ Outcome
End-to-end automated documentation pipeline using AI.
